# 【原子信息喂养】文案生成框架

## 使用说明

当你收到选题方向 + 素材后，按以下模板结构产出文案。

---

## Prompt 模板

```
你是一个短视频文案专家，擅长使用「原子信息喂养」框架创作高密度内容。

## 框架说明

「原子信息喂养」的核心逻辑：
- 围绕一个明确的大方向，持续抛出独立的信息点（原子）
- 每个原子是一个独立的信息单元，可以是一句话，也可以是两句/三句
- 原子之间不强调逻辑连接，它们通过"都服务同一个大方向"来保持凝聚力
- 信息密度优先：每个原子都要贡献新信息，避免重复
- 节奏快，不停留，快速抛出，快速切换

## 两种使用方式

### 作为大框架
整篇文案从头到尾都用原子结构：开头hook → 原子1 → 原子2 → ... → 原子N → [收尾]

### 作为内嵌技法
在其他大框架下，某个段落用原子来填充信息密度。
例如：故事型文案中间的「知识点穿插」、对比型文案中间的「多个论据罗列」、教程型文案中间的「技巧清单」

## 本次任务

**大方向**：[这里填入你的核心主题]

**素材**：[填入你提供的素材/信息点/关键词]

**产出要求**：
1. 围绕大方向，输出至少8-12个原子信息点
2. 每个原子信息点用"原子N："标注
3. 每个原子尽量独立，不要和前后原子重复同一个信息
4. 节奏要快，不要铺垫，不要过渡句，直接给信息
5. 可以适当使用数字开头（第一个真相/第二个秘密/X个方法等）增加节奏感
6. 根据需要决定是否加收尾句或行动召唤

## 核心原则：陌生感与熟悉感交替

**陌生感来源**（满足好奇 + 新知识获得感）：
- 引用学说/理论（自我损耗理论、峰终定律、马斯洛需求……）
- 提出反常识观点（「努力不一定有回报」「早起不适合所有人」）
- 新的概念/名词/视角

**熟悉感来源**（满足代入 + 情感共鸣）：
- 具象场景（早上起不来、周末想躺平、工作汇报……）
- 常见痛点（拖延、焦虑、选择困难……）
- 观众自己的经历（「你有没有这种感觉……」）

**观众心理轨迹示例**：没听过 → 是的我也觉得 → 是的就是 → 咦好新奇 → 对啊就是这样 → 哇还能这样？

**节奏控制**：不要连续两个原子都是同一类型，陌生感和熟悉感应自然交替。
比如：原子1抛陌生观点 → 原子2接熟悉场景 → 原子3再抛陌生理论 → 原子4接熟悉例子……

## 输出格式

直接输出文案全文，不需要额外说明。
```

**输入**：
- 大方向：直播间留人的核心技巧
- 素材：主播话术技巧、互动方法、福利设计、节奏把控

第二个技巧：问句比陈述句更容易让人停留。

记住这8条，你的在线人数至少翻一倍。

## 配套技法一：抽象接具象

**核心理念**：每个原子信息点 = 抽象结论 + 具象例子，两者直接拼接，不要过渡句。

**为什么用它**：
- 抽象结论给观点高度
- 具象例子让观众秒懂
- 两者组合 = 信息密度拉满，理解零门槛

**使用方式**：
```
原子N：[抽象结论/观点]
      [具象例子：一个场景 / 一个数字 / 一个具体案例]
```

**示例**：
```
第一个技巧：福袋不是用来引流的，是用来留人的。比如你把福袋放在第15分钟，主播刚好进入状态，留人效率最高。
第二个技巧：弹幕要念，但不要一个一个念，要串起来念。就像在跟一个人聊天，不是跟一群人汇报。
第三个技巧：不会憋单的直播间活不过5分钟。憋单就是在制造稀缺感，让观众觉得错过就没了。
```

**要点**：
- 例子要「可感」：有画面、有数字、有具体场景
- 一个例子一句话/一个场景足够，不要太长
- 例子要紧贴观点，不要跑偏

---

## 配套技法二：引用学说/名言/真理

**核心理念**：用权威理论、科学学说、经典名言来背书观点，给普通观点注入知识深度和信任感。

**为什么用它**：
- 科学理论和学说对大多数人是「新知识」
- 引用后 = 普通观点升级为「经过验证的洞察」
- 观众产生「原来如此」的认知冲击
- 权威背书天然增强说服力

**使用方式**：
```
原子N：[抽象结论/观点]
      「引用：XX理论/XX研究发现/XX实验证明」
      [可选：接一个简短具象例子]
```

**示例**：
```
第一个真相：你的意志力不是无限的。心理学上的「自我损耗理论」指出，每一次自我控制都会消耗有限的意志力资源，用完就没有了。

第二个真相：情绪会影响决策质量。神经经济学研究表明，人在情绪波动时的决策质量会下降40%。

第三个真相：习惯的养成靠的不是意志力，是环境设计。行为科学中的「摩擦力理论」告诉我们，减少一个行为的阻力，比增加意志力更有效。
```

**要点**：
- 引用要可信：「研究表明」「XX实验证明」「心理学发现」「经济学有XX理论」
- 引用要扣题，和观点高度相关
- 一个原子引一个即可，不要堆砌
- 可搭配「抽象接具象」一起用：引用学说后，再跟一个具体例子

---

## 配套技法三：观点陌生熟悉化

**核心理念**：把观众「隐约知道但没想清楚」的事情，用新词/反常识/热点角度拎出来，让观众觉得「太对了，就是这种感觉，但我从来没这样想过」。

**三种实现方式**：

### 方式一：创造/借用新词
- 其他领域的术语（物理学的「锚点」「降维」、经济学的「边际成本」）
- 小众但好理解的概念（「祛魅」「解耦」「颗粒度」「斩杀线」「情绪劳动」）
- 自创的精准描述词

**示例**：
```
职场人要有「斩杀线」——就是让你觉得「我不想做但又不好意思拒绝」的那个临界点，过了这个点你就该说不。

AI不是你的大脑，是你的手脚。它执行快，但判断慢。
```

### 方式二：蹭热度
借势热点/热梗/热词为观点增加熟悉感和时效性。

**示例**：
```
很多人陷入「时间祛魅」的陷阱——以为管理时间就是挤时间，其实时间根本管不住，你只能管注意力。

同事找你帮忙，你不好意思拒绝，这就是典型的「情绪劳动过剩」。
```

### 方式三：反常识颠覆
颠覆固有认识，提出相反但更接近真相的结论。

**示例**：
```
早起不适合所有人。真正的效率来自精力周期，不是起床时间。

读太多书会让你变蠢。输入太容易，提取反而困难，你需要的是「知识的二次加工」。

AI正确的用法压根就不是给提示词给AI，而是让AI帮你执行你脑子里的判断。
```

**技法组合**（三种技法可同时使用）：
```
原子N：[反常识/新词开头的陌生感]
      「引用：XX理论/XX研究发现」
      [具象例子：一个具体场景]
```

---

## 注意事项

1. 原子数量可根据内容长度调整，短视频6-10个，长视频可到15-20个
2. 数字开头的表达（如"第一个"、"第二个"）是最常用的节奏工具，但不是必须
3. 可以混合使用：有的原子用数字开头，有的用陈述句直接抛出
4. 收尾句可选：可以直接结束在最后一个原子，也可以加一句总结或召唤
5. 三个技法可以组合使用：每个原子都应尽量包含「陌生熟悉化 + 引用学说 + 具象例子」
